# sheep-n-ships-api

## How to run

1. Install dependencies:

```bash
yarn install
```

2. Start the database:

```bash
yarn mongo:start
```

3. Start the application:

```bash
yarn dev
```

## How to run tests

```bash
# unit tests
yarn test

# unit tests (watch mode)
yarn test:watch

# test coverage
yarn test:cov
```
